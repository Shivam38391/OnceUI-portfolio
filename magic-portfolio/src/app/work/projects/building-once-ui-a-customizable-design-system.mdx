## Overview

An AI-powered platform designed to simplify and automate expense management for businesses.

## Key Features

* **Effortless Submission:**
    * Employees can easily submit expenses by:
        * Snapping a photo of the receipt and sending it via WhatsApp (no separate app download needed).
        * Using email.
        * Accessing the web platform.
* **OCR Technology:**
    * Automatically extracts key data from receipts.
    * Categorizes expenses for better organization.
* **AI Fraud Detection:**
    * Identifies potential fraudulent activities by detecting:
        * Duplicate invoices.
        * Handwritten invoices.
        * Mismatched invoices.
        * High-risk vendors.
* **Admin Dashboard:**
    * Provides a centralized interface for employers to:
        * Review submitted expenses.
        * Approve or reject expenses.
        * Track expense status.
        * Generate comprehensive reports.
* **Automated Processing:**
    * Allows administrators to define policies for automatic approval of routine expenses, streamlining the workflow.
* **Multi-currency and Language Support:**
    * Designed to support global operations with:
        * Handling of multiple currencies.
        * Support for various languages.
* **Security:**
    * Prioritizes data security with features like:
        * Two-Factor Authentication (2FA) to protect sensitive information.

## Technologies Used

- **Next.js**: For fast, server-rendered React applications.
- **Figma**: For creating and managing design assets and prototypes.
- **ShadCN UI**: For styling React components with a modular, themable approach.
